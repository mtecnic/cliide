💡 Notable Design Decisions
============================

1. Textual Framework
   - Modern, well-maintained TUI library
   - CSS-like styling
   - Reactive architecture
   - Rich widget ecosystem

2. OpenAI SDK for VLLM
   - Compatible with VLLM's OpenAI endpoint
   - Mature, well-tested
   - Streaming support built-in

3. Pydantic for Configuration
   - Type-safe config
   - Validation
   - Environment variable support
   - JSON schema generation

4. Event-Driven Architecture
   - Loose coupling between components
   - Easy to extend
   - Testable

5. Hybrid LSP + AI
   - Best of both worlds
   - LSP for accuracy
   - AI for complex tasks
